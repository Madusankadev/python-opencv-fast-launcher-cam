<h1>Fast Launch CAM with Python OPEN-CV</h1>

<p>Pyhton Version: 3.12</p>
<p>Opencv Vesion: 4.13</p>

<h2>How to Run Poject</h2>

<p>First Clone the Repo in to Local machine</p>
<pre>git clone https://github.com/Madusankadev/python-opencv-fast-launcher-cam.git </pre>

<p>Then open terinal/cmd inside the project folder and first activate the virtual envirement</p>
<pre>py3.12/Scipts/activate</pre>

<p>Finally you can run script and launch your web cam</p>
<pre>py main.py</pre>

<p>Default inbult camera refers to 0. incase if you are using external camera change 0 to 1 or so on in line <pre>[cam = cv2.VideoCapture(0, cv2.CAP_DSHOW)</pre></p>
<p style="colo: red;">Pres 'q' to exit.</p>
